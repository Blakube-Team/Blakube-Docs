# Installation

### Requirements:
* A server running _PaperMC_ **1.20** or higher (lower versions not tested).
* _Java 21_ or higher installed on your server.
* Dependencies: PlaceholdersAPI.

## Installing BK-Tops
1. Download the latest version of BK-Tops.
2. Place the downloaded `.jar` file into your server's `plugins` directory.
3. Start or restart your server to generate the configuration files.


## Post-installation
1. After the first run, navigate to the `plugins/BK-Tops` directory.
2. Many folders and files will be generated. The folder _configuration_ contains files in order to customize general 
settings. For message customization and translation, check the _lang_ folder.
3. Don't forget to configure your tops in `tops.yml` following the [How to](/bk-tops/config/how-to) guide.
