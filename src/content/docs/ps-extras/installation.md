# Download
Purchase and download the latest plugin `.jar` file from the following **official** sites where PS-Extras is published:
- [BuiltByBit](https://builtbybit.com/resources/ps-extras-make-protections-awesome.79121/)

# Requirements
- A server running _PaperMC_ **1.20.1**, **1.20.3**, **1.20.4**, **1.20.5 to 1.21.8**. Other versions were not tested!
- _Java 21_ or higher installed on your server.
- Dependency: ProtectionStones (latest version)
- Soft-dependencies: [PlaceholdersAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)

# Installation
- Upload the downloaded `.jar` file to your serverâ€™s plugins folder.
- Restart your server!

# Configuration
- Once the config files for PS-Extras plugin are generated, navigate to the folder `..plugins/PS-Extras/`.
- Here you can find the configuration files to customize the plugin. Available: messages, guis and settings.
